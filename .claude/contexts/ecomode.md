# Ecomode Context

When ecomode is active, agents return compact output:
- Status: success | failed | partial
- Summary: 1-2 sentences max
- Key data: essential results only

Skip intermediate steps, verbose explanations, and full file contents.

## Activation
- 4+ parallel tasks spawned
- Batch operations
- Near compaction threshold (80%+ context)
- User command: "ecomode on"
