## 갱신 작업

### S3 결과 → 확정 사항 이동

각 미결 항목에 대해:

```markdown
### [항목명]
- **결정**: [선택된 방식]
- **근거**: [벤치마크 수치 또는 비교 결과 요약]
- **S3 참조**: [S3-0N 문서 링크]
```

### 잔여 미결 사항

S3에서 해소되지 않은 항목은:

```markdown
### [항목명]
- **현 상태**: [S3에서 어디까지 진행되었는지]
- **미해소 이유**: [시간 부족 / 추가 검증 필요 / 의존성 미충족]
- **다음 단계**: [실험 설계 단계에서 해소 / 추가 PoC 필요]
```

### 신규 확정 사항 추가

S2, S3에서 새로 확정된 사항도 "확정된 사항" 섹션에 추가:

- Agent Teams 워크플로 (S2-04)
- 평가 베이스라인 3종 (S2-01)
- 다양성 지표 5종 (S2-03)
