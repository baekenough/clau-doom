## VNC + noVNC 스트리밍

컨테이너 안에서 Xvfb(가상 디스플레이) → VizDoom(윈도우 모드) → x11vnc → noVNC(WebSocket → VNC 브릿지) → 브라우저에서 실시간 관전.

## 탭 구성

### Arena

```
┌─ Arena ──────────────────────────────────────┐
│  멀티 에이전트 Doom 화면 그리드 (noVNC)        │
│  AI 사고 과정 실시간 오버레이                   │
│  킬/데스/체력 실시간 지표                      │
└──────────────────────────────────────────────┘
```

### Player

```
┌─ Player ─────────────────────────────────────┐
│  개별 에이전트 상세 (전략 프로필, 실험 현황)     │
│  에이전트별 Cpk 게이지                         │
│  FMEA Top-5 실패 모드                         │
└──────────────────────────────────────────────┘
```

### Evolution

```
┌─ Evolution ──────────────────────────────────┐
│  세대별 진화 추이 차트                         │
│  X-bar/R 관리도 (SPC)                         │
│  전략 분포 변화 (PCA 2D 산점도)                │
│  학습 곡선 피팅 + plateau 예측                 │
│  TOPSIS 순위 변화                             │
└──────────────────────────────────────────────┘
```

### Research

```
┌─ Research ───────────────────────────────────┐
│  연구 일지 (RESEARCH_LOG.MD 실시간 렌더링)     │
│  실험 현황 (진행 중/완료/대기)                  │
│  DOE 매트릭스 시각화 (주효과도, 교호작용도)      │
│  ANOVA 결과 테이블, 잔차 진단 플롯             │
│  발견 누적 (FINDINGS.MD)                      │
│  메타 분석 (DOE 유형별 채택률, Phase별 학습률)   │
│  FMEA RPN 추이, 지식 베이스 성장 곡선          │
└──────────────────────────────────────────────┘
```
