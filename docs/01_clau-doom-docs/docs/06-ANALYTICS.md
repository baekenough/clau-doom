## 다변량 분석

### PCA (주성분 분석)

```
- 전략 파라미터 공간의 차원 축소
- 5~8개 파라미터가 실질적으로 2~3개 주성분으로 설명될 수 있음
- 주성분 축 해석 → "공격-방어 축", "탐색-활용 축" 같은 전략 차원 발견
- 이 차원이 세대 진화에 따라 어떻게 이동하는지 시각화 (논문 핵심 그림)
```

### 군집 분석 (Clustering)

```
- 에이전트 전략 프로필을 k-means/DBSCAN으로 군집화
- "전략 종(species)" 발견 → 진화 과정에서 전략 다양성 추이
- 세대가 진행되며 군집이 수렴/분화하는 패턴 관찰
```

## 회귀 모델링

```
DOE 결과를 다항 회귀로 피팅:
  y = β₀ + Σβᵢxᵢ + ΣΣβᵢⱼxᵢxⱼ + Σβᵢᵢxᵢ² + ε

적용:
  - 파라미터-성과 수학적 관계식 도출
  - 회귀식의 R² adj로 모델 적합도 평가
  - VIF(분산팽창인자)로 다중공선성 진단
  - 잔차 분석으로 모델 가정 검증
  - 최종 회귀식을 Rust 스코어링에 반영 가능
    → RAG 검색 없이 수학적 최적 행동 추론 경로
```

## 학습 곡선 분석

```
세대별 성과 향상을 학습 곡선으로 모델링:
  y = a · x^b  (멱법칙 학습 곡선)
  또는 y = a · (1 - e^(-bx))  (지수 포화 곡선)

적용:
  - plateau 도달 시점 예측 → PI가 전략 전환 타이밍 결정
  - Phase별 학습률 비교 → DOE 도입 전/후 학습 가속 정도 정량화
  - 논문: "DOE 적용 후 학습률 b가 X% 증가" — 핵심 실험 결과
```

## 시뮬레이션 최적화

```
진화 하이퍼파라미터 메타 최적화:
  - 교차율, 변이율, 엘리트 비율, 탐험 비율
  - LHS(Latin Hypercube Sampling)로 파라미터 공간 효율 탐색
  - 대리 모델(GP 또는 Random Forest) + EI 획득 함수
  - 또는 직교배열 기반 타구치 방법 적용
```
